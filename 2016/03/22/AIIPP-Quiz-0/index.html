Quiz 0

  1. Which of the following are syntactically correct strings?

    Try each of them in [CodeSkulptor][http://www.codeskulptor.org/].

    • “𝙸𝚝’𝚜 𝚊 𝚋𝚎𝚊𝚞𝚝𝚒𝚏𝚞𝚕 𝚍𝚊𝚢.”

    • ‘𝙷𝚎𝚕𝚕𝚘’

    • 𝙷𝚎𝚕𝚕o

    • “𝙷𝚎𝚕𝚕𝚘

    • “𝙷𝚎𝚕𝚕𝚘”

      The answer is "Hello" and 'Hello'.

  2. To display a value in the console, what Python keyword do you use?

    The answer is print.

  3. In the following code, there is one line starting with #. What does this line mean to Python?

    • This text is printed on the console.

    • This is a comment aimed at the human reader. Python ignores such comments.

    • This is a syntax error.

    • This text is used as a file name for the code.

    The answer is This is a comment aimed at the human reader. Python ignores such comments..

  4. Which of the following arithmetic expressions are syntactically correct?

    Try each of them in [CodeSkulptor][http://codeskulptor.org/].

    • 𝟻 - 𝟷 - 𝟹 - 𝟽 - 𝟶

    • 𝟻 * 𝟹 (𝟽 - 𝟸)

    • (𝟾 + (𝟷 + (𝟸 * 𝟺) - 𝟹))

    • 𝟹 ((𝟸 - 𝟿) + 𝟺)) (𝟸 + (𝟷 - 𝟹))

    • (𝟽 - 𝟸) / (𝟹 ** 𝟸)

    The answer is All.

  5. You would like to make it so that the variable 𝚘𝚞𝚗𝚌𝚎s has the value 16, thus representing one pound. What simple Python statement will accomplish this?

    • 𝟷𝟼 = 𝚘𝚞𝚗𝚌𝚎𝚜

    • 𝚘𝚞𝚗𝚌𝚎𝚜 := 𝟷𝟼

    • 𝚘𝚞𝚗𝚌𝚎𝚜 = 𝟷𝟼

    • 𝚘𝚞𝚗𝚌𝚎𝚜 == 𝟷𝟼

    The answer is 𝚘𝚞𝚗𝚌𝚎𝚜 = 𝟷𝟼.
